# Vero Sponsors

Vero is a free and open-source multi-node validator client. That will not change,
and everything available in Vero today will remain freely accessible under the MIT
license.

Vero Sponsors is a support tier for organizations that rely on Vero in production
and want to contribute to its ongoing maintenance and development.

## Why This Exists

Maintaining a production-ready validator client requires continuous work:

- implementing network upgrades
- developing new features
- supporting users
- maintaining reliability across releases

The Vero Sponsors tier represents a way to fund that work without restricting access
to the core software.

## What Sponsors Receive

Sponsors get access to additional capabilities and support intended for production use:

- [Exclusive features](features.md)

    Includes operational tooling and enhancements not part of the open-source version
    (e.g. simplified high-availability deployment options)

- Dedicated support

    Private support channel with 24/7 availability for critical incidents

- Feature requests

    Ability to propose and prioritize features based on real-world needs
